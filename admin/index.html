Script Tag

Paste this single line before the </body> tag on the client's website. That's all they need.

What it does

The script loads asynchronously — it won't slow down the client's page. On load, it fetches their brand config (colors, name, welcome message) from your backend and injects a floating chat bubble styled to match their brand. All conversations go through your backend — the client never needs to touch any code again.

WordPress / Webflow / Wix

For no-code platforms, use the Custom Code / Header-Footer plugin to inject it:

  • WordPress: Appearance → Theme Editor → footer.php, before </body>
  • Webflow: Project Settings → Custom Code → Footer Code
  • Wix: Settings → Advanced → Custom Code → Body - end
  • Shopify: Online Store → Themes → Edit Code → theme.liquid, before </body>
